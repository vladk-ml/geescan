GEEScan Dev Dashboard

System Status

GET /api/health - Combined health check of all system components
GET /api/health/api - Tests API server status
GET /api/health/db - Tests PostgreSQL database connection
GET /api/health/gee - Tests GEE service status
POST /api/auth/gee - Initializes Google Earth Engine authentication
GET /api/auth/gee/status - Verifies GEE authentication status

AOI Management

GET /api/aois - Retrieves all Areas of Interest
GET /api/aois/{id} - Retrieves a specific AOI by ID
DELETE /api/aois/{id} - Removes an AOI by ID

Create New AOI

POST /api/aois - Creates a new Area of Interest
Select an area to see its coordinates

Export Management

POST /api/export - Starts a GEE export task for the specified AOI and date range
GET /api/export/{task_id}/status - Checks the status of a GEE export task
